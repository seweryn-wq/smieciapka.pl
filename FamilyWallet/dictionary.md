### 1. Glossary of Terms (Actors and Systems)

Here we define all the key entities involved in the processes.

* **Actors:**
    * **User (Parent):** The main user of the application, managing the family's finances.
    * **Child:** A young user of the application who receives pocket money and learns to save.
    * **Financial Advisor:** A specialist providing financial consultations via the platform.
    * **Platform Admin:** The administrator managing the Family Wallet system, user accounts, and partners.
* **Locations:**
    * **Family Wallet Platform:** The central digital environment where all processes take place.
    * **Mobile Application:** The User interface on a smartphone for both Parent and Child.
    * **Advisor Panel:** A web application for financial advisors to manage clients and consultations.
    * **Partner Institution:** A financial institution (bank, brokerage house) whose system is integrated with Family Wallet.
* **Systems:**
    * **Family Wallet Central System (FWCS):** The main backend system, storing data and managing application logic.
    * **Bank Aggregation API:** A programming interface for integrating and retrieving data from external financial institutions (e.g., using PSD2).
    * **External Financial System (EFS):** An external partner’s IT system (e.g., a bank's system) from which transaction data is retrieved.
    * **Payment Module:** An external or internal system for handling subscriptions and payments for services.
* **Objects:**
    * **Family Financial Profile:** The User’s digital record, aggregating all their financial data.
    * **Transaction:** A single, digital record of a financial operation (e.g., card payment, transfer).
    * **Financial Consultation:** A video session or online chat with a Financial Advisor.
    * **Savings Goal:** A defined objective for which the family or child is saving money.
    * **Financial Alert:** An automatic notification generated by the system based on the analysis of User data (e.g., exceeding a budget).
    * **Subscription:** An active subscription plan (e.g., Freemium, Premium) defining the User’s scope of access to features.
